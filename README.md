# Niccolò Forte personal website

Static personal website prepared for GitHub Pages.

## Files

- `index.html` – page structure and content
- `style.css` – styling
- `script.js` – small mobile navigation script
- `assets/Niccolo-Forte-CV.pdf` – CV linked from the site
- `CNAME` – custom domain for GitHub Pages

## GitHub Pages setup

### Option A: user site
Create a public repository named:

`YOUR-USERNAME.github.io`

Then upload all files to the root of the repository.

### Option B: project site
Create any public repository, upload the files, and enable Pages from:

- Settings
- Pages
- Deploy from a branch
- Branch: `main`
- Folder: `/ (root)`

## Custom domain

This repo already includes a `CNAME` file for:

`niccoloforte.com`

At your domain registrar, you will still need to configure the DNS records for GitHub Pages.
