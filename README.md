# Niccolò Forte personal website

Updated static website for GitHub Pages.

## Files

- `index.html` – homepage
- `blog.html` – blog landing page
- `style.css` – site styling
- `script.js` – mobile navigation and scroll reveal animations
- `favicon.svg` – NF browser icon
- `assets/Niccolo-Forte-CV.pdf` – CV linked from the site
- `CNAME` – custom domain for GitHub Pages
- `.nojekyll` – ensures GitHub Pages serves the site as plain static files

## Upload notes

Replace the existing repo contents with these files, keeping them at the repository root.
GitHub Pages should stay configured to deploy from the `main` branch and `/ (root)`.
